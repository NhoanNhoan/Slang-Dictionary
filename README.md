# Slang-Dictionary
Slang dictionary

### Common Information:
**IDE:** Jetbrains IntelliJ IDEA Community Edition 2020.3.3

### Features:
* Find slang word.
* Find by definition.
* Show history(Not saved when reset the program).
* Insert a new slang word.
* Edit.
* Delete.
* Set to default.
* Random slang word.
* Quiz by slang word.
* Quiz by definition.
